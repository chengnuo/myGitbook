重复输出字符串 repeat a string

JavaScript实现-重复输出字符串 repeat a string

需求

重复输出字符串

（重要的事情说3遍）

重复一个指定的字符串 num次，如果num是一个负数则返回一个空字符串。

重复输出字符串 在 CodePen 中打开

var repeatString = function (str, num){
            let result = '';

            if(str == ''){
                result = '';
            }else{
                for(let i=0; i<num; i++){
                    result += str;
                }

            }
            return result;
        }

        console.log( repeatString('abc', 3) );
